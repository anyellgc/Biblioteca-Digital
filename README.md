# 📚 Biblioteca Digital

Proyecto desarrollado con Node.js, Express y MongoDB que permite a los usuarios registrarse, iniciar sesión y acceder a una biblioteca digital.

## 🚀 Tecnologías utilizadas

- Node.js
- Express.js
- MongoDB
- MongoDB Compass
- Multer
- HTML5
- CSS3
- JavaScript

## 👥 Roles del sistema

El sistema tiene dos tipos de usuarios:

### Usuario
- Registrarse
- Iniciar sesión
- Ver libros disponibles
- Descargar libros en PDF

### Administrador
- Iniciar sesión
- Subir libros en PDF
- Gestionar la biblioteca

⚠️ El sistema solo permite **2 administradores**.
